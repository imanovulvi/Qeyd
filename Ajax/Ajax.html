jQuery AJAX Temel Kullanımı

jQuery'de .ajax() metodu kullanılarak HTTP istekleri yapılabilir:

1. GET İsteği

Bir API'den veri almak için GET isteği yapabilirsiniz. type ayarı GET olaraq belirtildiyində, verilər belirtilən URL-dən çəkilir.


$.ajax({
    url: 'https://api.example.com/data',
    type: 'GET',
    success: function(data) {
        console.log(data);  // Gelen veriyi yazdır
    },
    error: function(error) {
        console.error('Xəta:', error);
    }
});

2. POST İsteği

Bir API'ye veri göndərmək üçün POST isteği yapılır. type ayarını POST olaraq belirleyin və göndərəceğiniz veriyi data içində tanımlayın.


$.ajax({
    url: 'https://api.example.com/data',
    type: 'POST',
    contentType: 'application/json',
    data: JSON.stringify({ name: 'John Doe', age: 30 }),  // Gönderilecek veri
    success: function(response) {
        console.log(response);  // Sunucudan gelen yanıt
    },
    error: function(error) {
        console.error('Xəta:', error);
    }
});

3. PUT İsteği

Bir verini güncəlləmək üçün PUT isteyi yapılabilir. type ayarını PUT olaraq belirleyin və güncellenməsi gereken veriyi data olaraq gönderin.


$.ajax({
    url: 'https://api.example.com/data/1',
    type: 'PUT',
    contentType: 'application/json',
    data: JSON.stringify({ name: 'Jane Doe', age: 25 }),  // Güncellenecek veri
    success: function(response) {
        console.log(response);  // Güncellenen veri
    },
    error: function(error) {
        console.error('Xəta:', error);
    }
});

4. DELETE İsteği

Bir verini silmək üçün DELETE isteyi yapabilirsiniz. type ayarını DELETE olaraq belirtmek yeterlidir.


$.ajax({
    url: 'https://api.example.com/data/1',
    type: 'DELETE',
    success: function(response) {
        console.log('Silme başarılı');
    },
    error: function(error) {
        console.error('Xəta:', error);
    }
});

$.get və $.post ilə Basitleştirilmiş Kullanım

GET və POST istekleri üçün .get() və .post() kimi qısayol metodlarını da istifadə edə bilərsiniz.


// GET isteği için
$.get('https://api.example.com/data', function(data) {
    console.log(data);
});

// POST isteği için
$.post('https://api.example.com/data', { name: 'John Doe', age: 30 }, function(response) {
    console.log(response);
});

Axios

Axios script etiketi ile doğrudan HTML dosyanıza ekleyebilirsiniz:

<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>

1. GET İsteği

Bir API-dən veri almak üçün axios.get istifadə edə bilərsiniz.


axios.get('https://api.example.com/data')
    .then(response => {
        console.log(response.data); // Gelen veri
    })
    .catch(error => {
        console.error('Xəta:', error);
    });

2. POST İsteği

Bir API'ye veri göndərmək üçün axios.post istifadə edə bilərsiniz.


axios.post('https://api.example.com/data', {
        name: 'John Doe',
        age: 30
    })
    .then(response => {
        console.log(response.data); // Sunucudan gelen yanıt
    })
    .catch(error => {
        console.error('Xəta:', error);
    });

3. PUT İsteği

Mevcut bir verini güncəlləmək üçün axios.put istifadə edilir.


axios.put('https://api.example.com/data/1', {
        name: 'Jane Doe',
        age: 25
    })
    .then(response => {
        console.log(response.data); // Güncellenen veri
    })
    .catch(error => {
        console.error('Xəta:', error);
    });

4. DELETE İsteği

Bir verini silmək üçün axios.delete istifadə edə bilərsiniz.


axios.delete('https://api.example.com/data/1')
    .then(response => {
        console.log('Silme başarılı');
    })
    .catch(error => {
        console.error('Xəta:', error);
    });

Ekstra Özellikler

Özel Başlıklar (Headers)

Bazı isteklerde özel başlıklar (headers) eklemeniz gerekebilir, örneğin yetkilendirme (authorization) işlemleri için.


axios.get('https://api.example.com/data', {
    headers: {
        'Authorization': 'Bearer token_value',
        'Content-Type': 'application/json'
    }
})
.then(response => {
    console.log(response.data);
})
.catch(error => {
    console.error('Xəta:', error);
});

Async/Await ilə Kullanım

Axios’u async/await ilə daha təmiz bir sintaksisle də istifadə edə bilərsiniz.


async function fetchData() {
    try {
        const response = await axios.get('https://api.example.com/data');
        console.log(response.data);
    } catch (error) {
        console.error('Xəta:', error);
    }
}

fetchData();

XMLHttpRequest (Köhnə Yöntem)

fetch API'si çıxmadan əvvəl yaygın olan XMLHttpRequest istifadə edilmektedir. Artıq daha az tercih edilse də, köhnə brauzerlər üçün hələ də dəstək sunabilir.


const xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.example.com/data', true);
xhr.onload = function() {
    if (xhr.status === 200) {
        console.log(JSON.parse(xhr.responseText));
    }
};
xhr.send();

Fetch API

fetch API, modern JavaScript'te asenkron HTTP istekleri göndermek üçün tercih edilen yöntemdir. JSON verilerini almak veya göndermek üçün olduqca kullanışlıdır.


fetch('https://api.example.com/data', {
    method: 'GET', // 'POST', 'PUT', 'DELETE' vb.
    headers: {
        'Content-Type': 'application/json'
    }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Xəta:', error));

1. GET İsteği

fetch ilə veri çəkmək üçün yalnız URL'i belirtmək kifayətdir. Varsayılan istek türü GET olduğundan əlavə ayarlara ehtiyac yoxdur.


fetch('https://api.example.com/data')
    .then(response => {
        if (!response.ok) throw new Error('Şəbəkə cavabı düzgün deyil');
        return response.json(); // Cavabı JSON formatına çevir
    })
    .then(data => console.log(data)) // Alınan verini konsola yazdır
    .catch(error => console.error('Xəta:', error)); // Xətaları tut

2. POST İsteği

Veri göndərmək üçün POST isteyi istifadə edilir. İstədiyiniz URL'i, metodunuzu və göndərəcəğiniz məlumatları qeyd edin.


fetch('https://api.example.com/data', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({ name: 'John Doe', age: 30 }) // Göndəriləcək veri
})
.then(response => {
    if (!response.ok) throw new Error('Şəbəkə cavabı düzgün deyil');
    return response.json();
})
.then(data => console.log(data))
.catch(error => console.error('Xəta:', error));

3. PUT İsteği

Mevcut bir verini güncəlləmək üçün PUT isteyi istifadə olunur. Güncellenəcək veri ilə birlikdə URL'i göstərmək kifayətdir.


fetch('https://api.example.com/data/1', {
    method: 'PUT',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({ name: 'Jane Doe', age: 25 }) // Güncellenəcək veri
})
.then(response => {
    if (!response.ok) throw new Error('Şəbəkə cavabı düzgün deyil');
    return response.json();
})
.then(data => console.log(data))
.catch(error => console.error('Xəta:', error));

4. DELETE İsteği

Bir verini silmək üçün DELETE isteyi istifadə olunur. Silinəcək verinin URL'i göstərilməlidir.


fetch('https://api.example.com/data/1', {
    method: 'DELETE'
})
.then(response => {
    if (!response.ok) throw new Error('Şəbəkə cavabı düzgün deyil');
    console.log('Silme işlemi başarılı');
})
.catch(error => console.error('Xəta:', error));

Sonuç

Yuxarıda göstərilən metodlarla jQuery, Axios və fetch API istifadə edərək HTTP istekleri göndərə bilərsiniz. Hər bir metodun özünəməxsus avantajları var, buna görə layihənizin ehtiyaclarına uyğun olanı seçməyiniz məsləhətdir.
